# Inclufy

Two tools that make job postings more accessible for neurodivergent people.

**For Employers** — Paste a job posting, see exclusionary language highlighted, fix it with one click.

**For Job Seekers** — Paste a job posting, get a simplified version with jargon translated and concerns flagged.

## How to use

Open `employers/index.html` or `job-seekers/index.html` in any browser. No install needed.

## How it works

Both tools share a pattern-matching engine (`patterns.js`) that checks for 6 categories of barriers:
- Excessive soft-skill requirements
- Jargon and ambiguity
- Over-specificity
- Complex application processes
- Sensory and social demands
- Missing inclusion signals

All processing happens in the browser. Nothing is stored or sent anywhere.
